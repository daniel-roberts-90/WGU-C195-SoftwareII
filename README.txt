2. On the login screen, enter a valid username and password that matches the information saved in the MySQL database
3. Once logged into the application, you will be presented with a Main Menu screen with the options for Appointments, Customers, Repots, and Log Out
4. Choose an option from the Main Menu to create, update, and delete data
